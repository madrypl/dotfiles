# screen - cheatsheet

Leader - Ctrl key
<C-Up> - next screen
<C-Down> - prev screen

# vim - cheatsheet
Leader - `

## cscope
<Leader><CR> - find symbol
<Leader>\ - find symbol definition

## other

<Leader>n - omni
<Leader><Up> - next location
<Leader><Down> - prev location
<Leader>] - find selected word
<Leader>a - toggle NERD tree
<Leader><Left> - window left
<Leader><Right> - window right
